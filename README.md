# Martin Calander Homebrew Tap

Homebrew tap for [Mac Drag Scroll](https://github.com/martincalander/MacDragScroll).

## Install

```sh
brew install --cask martincalander/tap/mac-drag-scroll
```

Mac Drag Scroll is currently unsigned and not Apple-notarized. On first launch, macOS may require opening it from Finder with right-click **Open**.

## Update

```sh
brew update
brew upgrade --cask mac-drag-scroll
```
